# Build Server
Requirements: 
- nodejs  

## Run Server
```
clone https://github.com/lolamtisch/Discord-RPC-Extension.git
cd Discord-RPC-Extension
npm install
node server.js
```

## Generate Executables
```
clone https://github.com/lolamtisch/Discord-RPC-Extension.git
cd Discord-RPC-Extension
npm install
npm run build
```
All executables can be found in the dist folder.
